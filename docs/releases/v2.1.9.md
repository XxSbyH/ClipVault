# ClipVault v2.1.9

## 更新内容

- 修复最近历史快捷键触发后只显示 HUD、但没有粘贴到前台应用的问题。
- 最近历史快捷键和固定内容快捷键在模拟 `Ctrl+V` 前会等待 `Ctrl`、`Alt`、`Shift`、`Win` 修饰键释放，避免 `Ctrl+Alt+1` 等触发组合污染后续粘贴快捷键。
- 保留 2.1.8 的自动更新、最近十条历史快捷键和托盘交互能力。

## 发布文件

- `ClipVault_2.1.9_windows_x64_setup.exe`：Windows x64 安装包。
- `ClipVault_2.1.9_windows_x64_portable.exe`：Windows x64 便携版，可直接运行，适合临时验证或不想安装的场景。
- `ClipVault_2.1.9_windows_x64_setup.exe.sig`：Tauri updater 签名文件。
- `latest.json`：自动更新元数据，供客户端检查新版本。
- `ClipVault_2.1.9_macos_universal.dmg`：macOS Universal 安装镜像。
- `ClipVault_2.1.9_macos_universal.app.zip`：macOS Universal app 压缩包，适合手动解压使用。

## 验证说明

- 自动更新完整链路需要使用已发布的 GitHub Release 验证 2.1.8 客户端能检测、下载、安装并重启到 2.1.9。
- 最近历史快捷键粘贴行为仍需要在真实 Windows 桌面会话中人工确认。
