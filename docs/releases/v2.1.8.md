# ClipVault v2.1.8

## 更新内容

- 新增最近十条历史记录快捷键，可在设置页为位置 1-10 单独录制和启用快捷键，位置 1 表示最新一条历史。
- 最近历史快捷键触发后会直接写入剪贴板并粘贴到当前前台应用，历史不足时不会误粘贴。
- 最近历史快捷键参与现有快捷键冲突检测，避免与常规快捷键和固定内容快捷键重复。
- 调整系统托盘交互：左键单击打开并聚焦主窗口，右键显示托盘菜单，右键菜单移除重复的打开入口。
- 新增 Tauri 自动更新支持：应用启动后检查 GitHub Release 更新，用户确认后下载、安装并重启。
- 新增 updater 签名配置和发布说明，发布包会生成安装包签名文件供自动更新校验。

## 发布文件

- `ClipVault_2.1.8_windows_x64_setup.exe`：Windows x64 安装包。
- `ClipVault_2.1.8_windows_x64_portable.exe`：Windows x64 便携版，可直接运行，适合临时验证或不想安装的场景。
- `ClipVault_2.1.8_windows_x64_setup.exe.sig`：Tauri updater 签名文件。
- `latest.json`：自动更新元数据，供客户端检查新版本。
- `ClipVault_2.1.8_macos_universal.dmg`：macOS Universal 安装镜像。
- `ClipVault_2.1.8_macos_universal.app.zip`：macOS Universal app 压缩包，适合手动解压使用。

## 验证说明

- 自动更新完整链路需要使用已发布的 GitHub Release 验证旧版本客户端能检测、下载、安装并重启。
- 最近历史快捷键、托盘左键/右键行为仍需要在真实 Windows 桌面会话中人工确认。
